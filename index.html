Install

Upload, connect, publish.

No local build chain is required for normal deployment. Download the release, upload it to hosting, connect the database, and finish the browser installer.

production setup

# Requirements
PHP 8.2+ / MySQL 5.7+ or MariaDB 10.3+ / Apache or LiteSpeed

# Download
https://github.com/Vondereich/VonCMS/releases

# Upload and install
unzip voncms-latest.zip -d public_html/
open https://yourdomain.com

# Done
database connected
admin account created
VonCMS is live
