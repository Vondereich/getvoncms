Installation

Up in 2 minutes.

No CLI. No Node.js dependency hell. Upload, extract, open browser, done.

bash · installation guide
# System Requirements: PHP 8.1+, MySQL/MariaDB, Apache/Nginx
 
STEP 1 → Download from GitHub Releases
wget https://github.com/Vondereich/VonCMS/releases
 
STEP 2 → Upload to your hosting (cPanel / DirectAdmin)
unzip voncms-latest.zip -d public_html/
 
STEP 3 → Open your domain in browser
open https://yourdomain.com
 
STEP 4 → Follow the 2-minute installation wizard
✓ Database connected
✓ Admin account created
✓ VonCMS is live.
